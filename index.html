Luh and Jay Project

This project was crafted for a client named Luh and Jay (Pty) Ltd, the client had an event coming up and who was seeking a system to sell his tickets due within a week. Sadly, the event has been postponed due to the Corona Virus. I used Wordpress to create the website and plugins such as Tickera and WooCommerce. The challenge I had was that the theme chosen by the client and ticket system were not compatible. We had to use jQuery, CSS and javaScript so that the tickets can still be displayed the way the clients prefers. It required me to change the theme using CSS and JQuery. Another challenge I had was that the front-end builder and the map malfunctioned. Fortunately, I was able to communicate with the theme developers who gladly assisted.
